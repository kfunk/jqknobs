Knobs

I love knobs. When I was creating this thing I wanted a knob interface like the real equipment. Couldn't find a plugin so I made one myself.

This plugin is real easy to use.
A div for the knob image, and a DOM element that can hold a value.
html:
<div id="knob"> <input type="text" id="knobVal" />

In the Javascript jQuery ready function, call the jqKnob passing two options the image name and path, and the element ID from above.
script:
$(function(){
     $('#knob').jqKnob({
       knbImgSrc: 'images/knob1.png',
       knbValID: 'knobVal'
     })
});

These knobs are just some quick images I through together to see what can be done. Each a differnt size square with a circle and a transparent backgound.
